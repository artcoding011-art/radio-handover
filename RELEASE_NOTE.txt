[라디오 업무인수인계 시스템 Release Note]

작성일 : 2026.04.02

본 문서에는 로컬 DB 기반 시스템에서 Supabase 클라우드 데이터베이스 연동으로 전환한 시점부터 현재까지의 모든 업데이트 내용을 정리하였습니다.

---

### 1. 데이터베이스 인프라 전환 (Supabase 연동)
* [DB 전환] 기존 로컬 파일(`local_db.json`) 기반 저장소에서 Supabase(PostgreSQL) 클라우드 DB로 메인 데이터 저장소 전환
* [동기화] 실시간 데이터 동기화 및 Vercel 배포 환경 최적화
* [안정성] 로컬 데이터 백업 기능을 폴백(Fallback)으로 유지하여 오프라인/에러 상황 대비

### 2. 레이아웃 및 디자인 고도화
* [너비 최적화] 업무인계서 및 제작일정 관리 영역을 가로 '800px' 중앙 정렬 레이아웃으로 고정 (가독성 및 안정감 확보)
* [UI 개선] 전체적인 폰트 크기 조정, 버튼 형태 개선, 다크톤 헤더 및 현대적인 컬러 시스템 적용 (Indigo/Emerald 테마)
* [결재 영역] 근무자/결재자 확인 영역 디자인 수정 및 중앙 정렬 최적화

### 3. 제작일정 관리 기능 강화 (Schedule Management)
* [관리 탭 분리] '주간 기본 편성(반복)'과 '단일 녹음 일정(일별 추가)' 관리를 별도 탭으로 분리
* [일괄 처리] 매체별(1R, 2R, MFM) 프로그램 일괄 체크(제작완료) 기능 추가
* [시각화] 달력에서 제작 완료된 날짜를 '연한 초록색 배경 + 체크 아이콘'으로 표시
* [편의성] 제작 완료된 항목에 취소선 적용 및 시각적 피드백 강화

### 4. 주요 버그 수정 및 최적화
* [에러 방어] JSON 데이터 누락 또는 손상 시 서버 중단 없이 자동 백업 및 초기화하는 방어 로직 구현(`lib/db.ts`)
* [계산 로직] 달력의 제작 완료 표시 계산 범위 확장 (최근 60일치 데이터 포함)으로 지난달 기록 누락 문제 해결
* [성능 개선] 불필요한 리렌더링 방지를 위한 낙관적 업데이트(Optimistic UI) 적용
